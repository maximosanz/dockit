Methods

This page gives information on the methods used to produce and score docking models.

Docking methods


Name Description Performance in CAPRI1,2 Link to web server Link to paper
ClusPro Docking using PIPER, a Fast Fourier Transform (FFT) method. Four types of scoring functions are available - balanced, electrostatic favoured, hydrophobic favoured and VdW+electrostatic. Hierarchical clustering is performed on the top 1000 structures and the centers of the x largest clusters given out, where x is 10 by default. 1st (CAPRI5), 1st (CAPRI4) http://cluspro.bu.edu/login.php http://onlinelibrary.wiley.com/doi/10.1002/prot.24403/abstract
Hex A FFT method. Each molecule is modelled using 3D expansions of real orthogonal spherical polar basis functions to encode both surface shape and electrostatic charge and potential distributions. 5th (CAPRI5) http://hex.loria.fr/ http://nar.oxfordjournals.org/content/38/suppl_2/W445
ZDOCK Shape complementarity, desolvation and electrostatics optimised using FFT. Target functions are highly-tolerant of conformational changes on binding. - http://zlab.umassmed.edu/zdock/ http://onlinelibrary.wiley.com/doi/10.1002/prot.10092/pdf
PATCHDOCK Based on object recognition and image segmentation techniques. Looks for complementary shapes by identifying concave and convex patches and fitting them together. Ranks models based on geometric complementarity. 11th (CAPRI5) http://bioinfo3d.cs.tau.ac.il/PatchDock/ http://www.ncbi.nlm.nih.gov/pmc/articles/PMC1160241/?tool=pubmed
FIBERDOCK Backbone flexibility is modelled by an unlimited number of normal modes, hence accounting for small scale conformational changes. The structures of flexible proteins are iteratively minimised along the most relevant nodes. 6th (CAPRI5), 2nd (CAPRI4) http://bioinfo3d.cs.tau.ac.il/FiberDock/ http://onlinelibrary.wiley.com/doi/10.1002/prot.22668/full
FIREDOCK Each rigid-body input is refined by restricted interface side-chain rearrangement and by soft rigid-body optimization. The relative position of the docking partners is refined by Monte Carlo minimization of the binding score function, which is used to rank solutions. 3rd (CAPRI4) http://bioinfo3d.cs.tau.ac.il/FireDock/ http://www.ncbi.nlm.nih.gov/pubmed/18424796
Rosetta A low-resolution, rigid-body, Monte Carlo search is followed by simultaneous Monte Carlo minimization of backbone displacement and side-chain conformations. - http://rosie.rosettacommons.org/ http://graylab.jhu.edu/publications/2003JMB.pdf

Scoring methods


Name Description Performance in Moal et al. (2013)3 Link to paper
ZRANK Uses detailed electrostatics, van der Waals and desolvation terms with optimised weightings. 34th http://www.ncbi.nlm.nih.gov/pubmed/17373710
ZRANK2 Re-optimised ZRANK function. 1st http://www.ncbi.nlm.nih.gov/pubmed/18214977
PISA Combines coarse-grained and atomic potentials. 8th http://www.ncbi.nlm.nih.gov/pubmed/23180599
SPIDER Transforms protein-protein complexes into a residue contact network then uses a subgraph mining approach to identify common interfacial residue patterns that appear in at least a subset of native protein-protein interfaces. 38th http://www.ncbi.nlm.nih.gov/pubmed/22581643
